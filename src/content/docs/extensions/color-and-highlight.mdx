<Step>Run the following command to install associated toolbar component.</Step>
```bash
npx shadcn add https://tiptap.niazmorshed.dev/r/color-and-highlight.json
```

## Usage

```jsx
const extensions = [
	StarterKit.configure({
		// ...
	}),
	TextStyle,
	Color,
	Highlight.configure({
		multicolor: true,
	}),
];

const editor = useEditor({
	extensions,
});
```
