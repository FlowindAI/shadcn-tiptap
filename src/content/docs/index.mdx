## Overview

This project provides:

1. A set of custom-built extensions for Tiptap
2. Ready-to-use toolbars crafted with [shadcn/ui](https://ui.shadcn.com/) components
3. Easy integration using the [shadcn CLI](https://ui.shadcn.com/docs/cli)



## Getting Started

To get started with shadcn-tiptap, please refer to [Installation Guide](/docs/installation).
